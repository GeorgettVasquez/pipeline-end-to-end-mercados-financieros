# Dise-o-e-implementaci-n-de-un-pipeline-de-precios-de-acciones-y-datos-macroeconomicos
El proyecto consiste en el diseño e implementación de un pipeline de datos End-to-End para la ingesta, almacenamiento, procesamiento y visualización de precios de acciones bursátiles combinados con indicadores económicos geoespaciales por región o país
